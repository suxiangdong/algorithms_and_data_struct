## 数学函数
算法分析中最常用的有七种函数，本文将分别介绍其概念以及定理，并附带部分练习题用以加深理解。

#### 常数函数
<img src="http://latex.codecogs.com/gif.latex?\\f(n) = c"></img>  
最简单的就是常数函数了！n是什么并不重要，f(n)的结果总是c。
