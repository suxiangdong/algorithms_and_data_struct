## 递归算法的不足
递归是一种很很强大的工具，但是十分容易误用。

在上一节中斐波那契数列的递归函数，就是一种极其糟糕的实现，不仅仅时间复杂度高大O(2^n)，
而且每一次递归调用都需要保存上下文环境，将返回的结果再次进行计算，这也会严重影响代码执行的效率。

## 尾递归
定义：如果每次递归调用的返回值可以立即返回，不需要额外的计算，那么这个递归就是一个尾递归。
